# Импорт карточек из YouTube — настройка

Фича: в любой обычной папке появилась кнопка **«Карточки из YouTube»**. Вставляешь ссылку на ролик до 20 минут, выбираешь «Слова / Фразы / Слова + фразы» — приложение достаёт транскрипт, выделяет лексику с переводом на русский, отбрасывает то, что уже есть во встроенных паках (A0/A1/A2/phrases) и в ранее импортированных YouTube-карточках, и показывает превью с галочками. Отмеченное становится карточками текущей папки; в описании карточки — уровень CEFR, часть речи и кликабельный таймкод на момент в видео.

## Архитектура

```
Браузер (js/screens/folder/youtube-dialog.js)
   │  POST /api/yt-video {url}
   ▼
netlify/functions/yt-video.mjs ──► Supadata API
   │   • GET /v1/youtube/video   — заголовок + длительность (лимит 20 мин)
   │   • GET /v1/transcript?mode=auto — субтитры; если их нет, Supadata сам
   │     делает AI-расшифровку (тогда клиент опрашивает ?jobId=…)
   ▼
Браузер: POST /api/yt-generate {title, lang, mode, segments}
   ▼
netlify/functions/yt-generate.mjs ──► Gemini generateContent (JSON-ответ)
   ▼
Браузер: сверка с паками и YouTube-папками (js/lib/youtube-import.js) → превью → store.createCard
```

Ни один API-ключ в браузер не попадает — оба живут в переменных окружения Netlify.

## Регистрация (обе бесплатные)

1. **Supadata** — транскрипты YouTube.
   - Зарегистрируйся на <https://supadata.ai>, в дашборде возьми API key.
   - Бесплатный тариф даёт стартовый пул кредитов: метаданные — 1 кредит, готовые субтитры — 1 кредит, AI-расшифровка — 2 кредита/мин. Это и есть фактический лимит обработанных роликов.
2. **Google AI Studio** — генерация карточек.
   - На <https://aistudio.google.com> создай API key (free tier).
   - Бесплатные лимиты Gemini Flash (порядка сотен запросов в день) для личного использования более чем достаточны: 1 ролик = 1 запрос.

## Переменные окружения Netlify

Site configuration → Environment variables:

| Переменная | Обязательна | Значение |
|---|---|---|
| `SUPADATA_API_KEY` | да | ключ из дашборда Supadata |
| `GEMINI_API_KEY` | да | ключ из Google AI Studio |
| `GEMINI_MODEL` | нет | по умолчанию `gemini-2.5-flash`; можно указать другую модель |

После добавления переменных нужен redeploy сайта. Функции подхватываются из `netlify/functions/` автоматически (`[functions]` в `netlify.toml`).

## Локальная разработка

```bash
npm i -g netlify-cli      # если ещё не стоит
netlify dev               # поднимает статику + функции на одном порту
```

`netlify dev` подтянет переменные окружения из привязанного сайта (`netlify link`) или из локального `.env` (не коммитить!).

## Ограничения и поведение при ошибках

- Ролики длиннее 20 минут отклоняются до траты кредитов на транскрипт (проверка по метаданным).
- Видео без субтитров: Supadata делает AI-расшифровку — это дольше (клиент ждёт до 3 минут) и дороже по кредитам.
- Приватные/удалённые видео, исчерпанные квоты, некорректный JSON от LLM — во всех случаях диалог возвращает понятное сообщение и позволяет попробовать снова.
- Netlify free: синхронная функция ограничена ~10 с. `yt-generate` укладывается за счёт компактного транскрипта (≤28k символов) и отключённого «размышления» модели; если начнут появляться таймауты — поставь `GEMINI_MODEL=gemini-2.5-flash-lite`.
- Эндпоинты `/api/*` публичны (как и сайт). Квоту защищают лимиты самих сервисов; при желании можно добавить проверку Supabase-JWT в функциях.

## Как определяются «уже известные» слова

- Все четыре встроенных пака (`packs/*.json`) — целиком, независимо от того, установлены ли они.
- Карточки текущей папки — целиком.
- Карточки остальных обычных папок — только созданные импортом из YouTube (признак: таймкод-ссылка `youtube.com/watch?v=` в описании).
- Сравнение — по нормализованной базовой форме: `walking/walked/walks` считаются формами `walk` (см. `stemVariants` в `js/lib/youtube-import.js`).
- Если LLM вернул и фразу, и входящие в неё слова — остаётся только фраза.

## Файлы фичи

| Файл | Роль |
|---|---|
| `netlify/functions/yt-video.mjs` | метаданные + транскрипт (Supadata), лимит 20 мин, polling AI-расшифровки |
| `netlify/functions/yt-generate.mjs` | промпт + вызов Gemini, валидация JSON-ответа |
| `js/lib/youtube-import.js` | чистые утилиты: парсинг ссылки, нормализация, сверка, description |
| `js/screens/folder/youtube-dialog.js` | диалог: форма → прогресс → превью → создание карточек |
| `css/screens/youtube-import.css` | стили диалога |
| `tests/youtube-import.test.js` | vitest-тесты утилит |

Изменённые: `js/screens/folder/index.js` (кнопка), `js/ui/constants.js` (иконка), `css/style.css` (@import), `sw.js` (precache, версия `kar-v12.3`), `netlify.toml` (`[functions]`).
